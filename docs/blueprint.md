# **App Name**: Editroy

## Core Features:

- QR Code Login: Users log in by scanning a QR code, granting access to document sending and receiving features. One QR code is valid for one login session only.
- Document Queue: Documents sent via QR code are queued for review and further processing within the application.
- Document Tools Suite: A suite of tools for PDF, Word, Excel, PowerPoint, and Image files, including converting, merging, splitting, compressing, and editing.
- Print Mode Preview Tool: Offers a print preview feature to simulate A4, A5, Legal, Letter sizes with configurable margins, ensuring the user does not unintentionally include information or settings which they would have preferred to omit.
- Document Mistake Checker: Automatically checks uploaded documents for grammatical errors and typos, improving document quality. This feature uses a tool to improve results by iteratively improving until it determines it has exhausted obvious remedies.
- Integrated Document Viewer: Built-in previewer to view documents before print/save. Restricts direct downloads until previewed and approved.
- ID Card Copier Tool: Upload front/back of ID -> auto-fit on single A4 page (duplicate if needed).
- Multiple Pages per Sheet (N-up Printing): Allow printing 2, 4, 8 pages on one A4 sheet for saving paper.
- Page Selection & Range Tool: User selects which pages to print (e.g., 2-5, 7, 10-12).
- Black & White or Color Toggle: Choose printing mode for each file.
- Duplex (Double-sided) Toggle: Choose single or double-side printing, auto-arrange pages.
- Passport Photo Generator: Upload one image -> auto crop to passport size in multiple formats (2x2, 35x45 mm, etc.).

## Style Guidelines:

- Primary color: Soft coral red (#F08080) for a clean and trustworthy feel.
- Background color: Light salmon (#FFA07A) for a professional and neutral backdrop.
- Accent color: Muted brick red (#B22222) to highlight important actions and buttons.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral look, suitable for both headlines and body text.
- Use a consistent set of minimalist icons throughout the application to represent document types and actions.
- A clean, intuitive layout with clear visual hierarchy. Important functions should be easily accessible.
- Subtle transitions and animations for document loading and tool activation to provide feedback to the user.
